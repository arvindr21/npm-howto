Steps to Create your Node Module


  • Step 1 : Create and clone a git hub repository
  • Step 2 : Create a package.json and include dependencies
  • Step 3 : Create test cases for your code
  • Step 4 : Create an account with NPM (one time) and publish your module

Step - 1


 Create and clone a git hub repository
  • Go to https://github.com/
  • Create a new account
  • Create a new repositoryhttps://github.com/new
  • Get the git clone url (Should be on the right hide side of the project main page) and clone it to a folder (myModule)
    • ex:git@github.com:arvindr21/npm-calculator.git

Step - 2


Create a package.json and include dependencies
  • Typically to build a node module, you need Node as well as npm
  • Head to http://nodejs.org/ and click on the shiny green Install  button - the set up is fairly simple
  • Then open a terminal/command prompt and run
    npm -v  to verify the installation
  • Next, open folder: myModule, open/navigate your terminal/prompt here & execute npm init. This will ask a set of questions and Bam!! you have your package.json file


PS: We are creating a calculator node module
Sample package.json ->

Package.json

{
  "name": "npm-calculator",
  "version": "1.0.1",
  "description": "This is a demo npm-calculator",
  "main": "index.js",
  "scripts": {
    "test": "jasmine-node specs/"
  },
  "repository": {
    "type": "git",
    "url": "https://github.com/arvindr21/npm-calculator.git"
  },
  "keywords": [
    "calculator"
  ],
  "author": "Arvind Ravulavaru",
  "license": "MIT",
  "bugs": {
    "url": "https://github.com/arvindr21/npm-calculator/issues"
  },
  "homepage": "https://github.com/arvindr21/npm-calculator",
  "devDependencies": {
    "jasmine-node": "~1.13.1"
  }
}

Step - 2

Create a file (index.js) at the root of the folder, and add the below code to index.js. This is our node module.
exports.add = function(a,b)
{
	return a+b;
};
	
exports.subtract = function(a,b)
{
	return a-b;
};
	
exports.multiply = function(a,b)
{
	return a*b;
};

exports.divide = function(a,b)
{
	return a/b;
};
	
exports.modulo = function(a,b)
{
	return a%b;
};

Step - 3


 Create Test cases and validate our module
  • As you can see from the sample package.json earlier, we have added a "devDependencies" section
  • Lets set that up. Back to terminal/prompt and run "npm install"
  • As you can see, this has added our jasmine-node testing framework to our module
  • Create a folder called "specs" at the root and add a file called "calculator-spec.js"
  • This is where our test cases reside

Step - 3

Jasmine test cases for the calculator module. Copy the below to calculator-spec.js
var calculator = require("../index");
 
describe("multiplication", function () {
 // Add 2 numbers
  it("should add 2 and 3", function () {
    var sum = calculator.add(2, 3);
    expect(sum).toBe(5);
  });
 
 
  // Subtract 2 numbers
  it("should Subtract 2 and 3", function () {
    var difference = calculator.subtract(2, 3);
    expect(difference).toBe(-1);
  });
 
  // Multiply 2 numbers
  it("should Multiply 2 and 3", function () {
    var product = calculator.multiply(2, 3);
    expect(product).toBe(6);
  });
 
  // Divide 2 numbers
  it("should Divide 6 and 2", function () {
    var reminder = calculator.divide(6, 2);
    expect(reminder).toBe(3);
  });
 
 
  // Modulo 2 numbers
  it("should Modulo 6 and 2", function () {
    var quot = calculator.modulo(6, 2);
    expect(quot).toBe(0);
  });
 
 
 
 });

Step - 3


 Execute the created test cases
  • Back to terminal/prompt, execute "npm test"
  • This will let you know, if everything is fine or not. Ideally it should say
    "5 tests, 5 assertions, 0 failures, 0 skipped"

Step - 4


 Create a npmjs account
  • Go to https://npmjs.org/signup and create an account
  • Come back to the terminal/prompt and execute
    "npm adduser"
    provide your credentials and then
    "npm publish"
  • You can go to https://npmjs.org/~ and your package will be listed at the bottom of the page. (login if needed)


"Bam!!"

Steps to use the created NPM Module

  • Step 1 : Create a folder in which you want to install the NPM Module
  • Step 2 : Install Node (if not already done)
  • Step 3 : execute 'npm init' -  Follow the steps above to create a package.json
  • Step 4 : Add the package name as a dev dependency
  • Step 5 : Execute 'npm install'
  • Step 6 : Leverage the library for your application

    • PS: If you are testing your package, create a new folder called "myCalcApp" and follow the above & next steps. Update your terminal/prompt path too.

    Sample package.json & myApp.js ->

Package.json


{
  "name": "myCalcApp",
  "version": "0.0.1",
  "description": "This is a calculator app that leverages the npm-calculator module to execute a few functions",
  "main": "myApp.js",
  "author": "Arvind Ravulavaru",
  "license": "MIT",
  "devDependencies" : {
  	"npm-calculator" : "latest"
  }
}

myApp.js


Create a file at the root and name it myApp.js. The add the below code.
var calculator = require("npm-calculator");
console.log(calculator.add(5,4));
  • Save both the files and the run "npm install" to install the calculator app and its dependencies
  • Then execute "node myApp.js"
  • The output would be
9

"Bam!!"
